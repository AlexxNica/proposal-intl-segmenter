Intl.Segmenter Proposal

1Segmenter Objects#

1.1The Intl.Segmenter Constructor#

The Segmenter constructor is a standard built-in property of the Intl object.

1.1.1Intl.Segmenter ([ locales [ , options ]])#

When the Intl.Segmenter function is called with optional arguments the following steps are taken:

  1. If NewTarget is undefined, let newTarget be the active function object, else let newTarget be NewTarget.
  2. Let segmenter be ? OrdinaryCreateFromConstructor(newTarget, %SegmenterPrototype%).
  3. Let requestedLocales be ? CanonicalizeLocaleList(locales).
  4. Let opt be a new Record.
  5. Let matcher be ? GetOption(options, "localeMatcher", "string", « "lookup", "best fit" », "best fit").
  6. Set opt.[[localeMatcher]] to matcher.
  7. Let r be ResolveLocale(%Segmenter%.[[AvailableLocales]], requestedLocales, opt, %Segmenter%.[[RelevantExtensionKeys]]).
  8. Set segmenter.[[Locale]] to the value of r.[[Locale]].
  9. If options is undefined, then
    1. Let options be ObjectCreate(%ObjectPrototype%).
  10. Else
    1. Let options be ? ToObject(options).
  11. Let granularity be ? GetOption(options, "granularity", "string", « "grapheme", "word", "sentence", "line" », "grapheme").
  12. Set segmenter.[[SegmenterGranularity]] to granularity.
  13. If granularity is "line",
    1. Let strength be ? GetOption(options, "strength", "string", « "strict", "normal", "loose" », "normal").
    2. Set segmenter.[[SegmenterStrength]] to strength.
  14. Return segmenter.

1.2Properties of the Intl.Segmenter Constructor#

The Intl.Segmenter constructor has the following properties:

1.2.1Intl.Segmenter.prototype#

The value of Intl.Segmenter.prototype is %SegmenterPrototype%.

This property has the attributes { [[Writable]]: false, [[Enumerable]]: false, [[Configurable]]: false }.

1.2.2Intl.Segmenter.supportedLocalesOf (locales [, options ])#

When the supportedLocalesOf method of %Segmenter% is called, the following steps are taken:

  1. Let availableLocales be %Segmenter%.[[AvailableLocales]].
  2. Let requestedLocales be ? CanonicalizeLocaleList(locales).
  3. Return ? SupportedLocales(availableLocales, requestedLocales, options).

The value of the length property of the supportedLocalesOf method is 1.

1.2.3Internal slots#

The value of the [[AvailableLocales]] internal slot is implementation defined within the constraints described in 1.4.

The value of the [[RelevantExtensionKeys]] internal slot is [].

Note CLDR defines several extension keys, but this specification does not expose them.

1.3Properties of the Intl.Segmenter Prototype Object#

The Intl.Segmenter prototype object, referred to as %SegmenterPrototype%, is itself an ordinary object. It is not a Segmenter instance, and has no instance-like internal slots.

1.3.1Intl.Segmenter.prototype.constructor#

The initial value of Intl.Segmenter.prototype.constructor is %Segmenter%.

1.3.2Intl.Segmenter.prototype.segment( string )#

When the Intl.Segmenter.prototype.segment is called with an argument string, the following steps are taken:

  1. Let segment be this value.
  2. If Type(segment) is not Object or segment does not have an [[SegmenterGranularity]] internal slot, throw a TypeError exception.
  3. Let string be ? ToString(string).
  4. Return ? CreateSegmentIterator(segment, string, forwards).

1.3.3Intl.Segmenter.prototype.reverseSegment( string )#

When the Intl.Segmenter.prototype.reverseSegment is called with an argument string, the following steps are taken:

  1. Let segment be this value.
  2. If Type(segment) is not Object or segment does not have an [[SegmenterGranularity]] internal slot, throw a TypeError exception.
  3. Let string be ? ToString(string).
  4. Return ? CreateSegmentIterator(segment, string, backwards).

1.3.4Intl.Segmenter.prototype.resolvedOptions ()#

This function provides access to the locale and options computed during initialization of the object.

The function returns a new object whose properties and attributes are set as if constructed by an object literal assigning to each of the following properties the value of the corresponding internal slot of this Segmenter object. (see 1.4): locale, granularity and strength. Properties whose corresponding internal slots are not present are not assigned.

1.4Internal slots of Intl.Segmenter Instances#

Objects that have been successfully initialized as a Segmenter object also have several internal slots that are computed by the constructor:

  • [[Locale]] is a String value with the language tag of the locale whose localization is used by the segmenter.
  • [[SegmenterGranularity]] is one of the String values "grapheme", "word", "sentence", or "line", identifying the segmenter used.
  • [[SegmenterStrength]] is present on Segmenter instances with line granularity. If present, it may be "strict", "normal", or "loose".

1.5Segment Iterators#

The methods Intl.Segment.prototype.segment and Intl.Segment.prototype.reverseSegment return iterators over the segments for a particular string. This section describes those iterator objects.

1.5.1CreateSegmentIterator ( segmenter, string, direction )#

When the abstract operation CreateSegmentIterator is called with Segmenter segmenter, string string and a direction either forwards or backwards direction, the following steps are taken:

  1. Let iterator be ObjectCreate(%SegmentIteratorPrototype%).
  2. Let iterator.[[SegmentIteratorSegmenter]] be segmenter.
  3. Let iterator.[[SegmentIteratorString]] be string.
  4. Let iterator.[[SegmentIteratorDirection]] be direction.
  5. If direction is forwards,
    1. Let iterator.[[SegmentIteratorPosition]] be 0.
  6. Else,
    1. Assert: direction is backwards.
    2. Let iterator.[[SegmentIteratorPosition]] be Get(string, "length").
  7. Let iterator.[[SegmentIteratorBreakType]] be an implementation-dependent string representing a break at the edge of a string.

1.5.2AdvanceSegmentIterator ( iterator )#

When the abstract operation AdvanceSegmentIterator is called with a segment iterator iterator, the following steps are taken:

  1. Let segmenter be iterator.[[SegmentIteratorSegmenter]].
  2. Let string be iterator.[[SegmentIteratorString]].
  3. Let direction be iterator.[[SegmentIteratorDirection]].
  4. Let position be iterator.[[SegmentIteratorPosition]].
  5. If direction is forwards and position is the length of string, or if direction is backwards and position is 0, return true.
  6. Based on the algorithm in UTS 14 or UTS 29, find the next or previous (based on direction) break in string from the position position based on the locale, granularity and strength in segmenter.[[Locale]], segmenter.[[SegmenterGranularity]] and, if present, segmenter.[[SegmenterStrength]].
  7. Set iterator.[[SegmentIteratorBreakType]] to a string representing the type of break found.
  8. Set iterator.[[SegmentIteratorPosition]] to the position of the newly found break point.
  9. Return false.

1.5.3%SegmentIteratorPrototype%#

%SegmentIteratorPrototype% is the prototype of all segment iterators. %SegmentIteratorPrototype% has the following properties:

1.5.3.1%SegmentIteratorPrototype%.next( )#

  1. Let iterator be this value.
  2. If iterator does not have a [[SegmentIteratorSegmenter]] internal slot, throw a TypeError exception.
  3. Let _previousPosition be iterator.[[SegmentIteratorPosition]].
  4. Let done be AdvanceSegmentIterator(iterator).
  5. If done is true, return CreateIterResult(undefined, true).
  6. Let newPosition be iterator.[[SegmentIteratorPosition]].
  7. Let string be iterator.[[SegmentIteratorString]].
  8. If iterator.[[SegmentIteratorDirection]] is forwards,
    1. Let segment be the substring of string from previousPosition to newPosition, inclusive of previousPosition and exclusive of newPosition.
  9. Else,
    1. Let segment be the substring of string from newPosition to previousPosition, inclusive of newPosition and exclusive of previousPosition.
  10. Let breakType be iterator.[[SegmentIteratorBreakType]].
  11. Let result be ! ObjectCreate(%ObjectPrototype%).
  12. Perform ! CreateDataProperty(result "segment", segment).
  13. Perform ! CreateDataProperty(result, "breakType", breakType).
  14. Return CreateIterResult(result, false).

1.5.3.2%SegmentIteratorPrototype%.advance( )#

  1. Let iterator be this value.
  2. If iterator does not have a [[SegmentIteratorSegmenter]] internal slot, throw a TypeError exception.
  3. return AdvanceSegmentIterator(iterator).

1.5.3.3get %SegmentIteratorPrototype%.position#

  1. Let iterator be this value.
  2. If iterator does not have a [[SegmentIteratorSegmenter]] internal slot, throw a TypeError exception.
  3. Return iterator.[[SegmentIteratorBreakType]].

1.5.3.4get %SegmentIteratorPrototype%.breakType#

  1. Let iterator be this value.
  2. If iterator does not have a [[SegmentIteratorSegmenter]] internal slot, throw a TypeError exception.
  3. Return iterator.[[SegmentIteratorBreakType]].
